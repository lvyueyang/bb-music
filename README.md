# 基于 B 站 API 作为歌曲源开发的音乐播放器

## 实现思路

1. B 站上有很多的音乐视频，相当于一种超级全的音乐聚合曲库（索尼直接将 B 站当做网盘，传了 15w 个视频）
2. 对这些视频进行收集制作成歌单
3. 无需登录即可完整播放，并且没有广告
4. 使用 [SocialSisterYi](https://github.com/SocialSisterYi/bilibili-API-collect) 整理的 B 站接口文档，直接就可以获取和搜索 B 站视频数据

## 功能

- [x] 播放器
  - [x] 基础功能
  - [x] 播放列表
  - [x] 单曲循环,列表循环,随机播放
  - [ ] 计时播放
  - [ ] 进度拖动
- [x] 搜索
  - [x] 视频名称关键字搜索
  - [ ] 按 UP 主搜索
  - [ ] 按 B 站 URL 搜索
- [ ] 歌单
- [ ] 歌单同步
- [ ] 歌单广场（由用户贡献分享自己的歌单）

## 技术栈

1. Next.js 作为全栈脚手架
2. React
3. zustand 状态管理
